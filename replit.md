# Bootcamp Projects

## Overview
This is a collection of bootcamp learning projects. Currently contains an R data transformation script demonstrating dplyr usage.

## Project Structure
- `project/` - Contains R scripts for data analysis
  - `data-transformation.r` - dplyr data transformation example
- `pdf/` - Documentation and notes

## Technologies
- R 4.5 with dplyr package

## Running the Project
The R script in `project/data-transformation.r` is a template for data transformation. To run it, you need to:
1. Load your data into a variable called `data`
2. Run the script with `Rscript project/data-transformation.r`
